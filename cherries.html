Lipslut / Cherries

Ecommerce website for monthly subscription earrings


React-driven, Gatsby-powered, Netlify-deployed

As part of a month-long internship, I worked in a team of 5 to build the website for Cherries from the ground up. Cherries is a new product line to be released by cosmetics and social activism platform, Lipslut, LLC. It focuses on providing customers refreshing and quality accessory experiences through monthly subscriptions for earrings.

Trending Technologies

Working on Cherries introduced a swath of new and trending technologies and API's. The website is entirely React-based, and built on static-site generator Gatsby. We used headless CMS Contentful to manage the product content, accessed through query language GraphQL. Finally, the website was deployed on hosting platform Netlify


Signup/Login User Account

Due to the serverless nature on Gatsby and Netlify, I utilized AWS Lambda functions for user account signup and login. I also maintained logged-in token to access user information on profile and orders through React's Context API.


Shopping Cart & Checkout

Users can add multiple products to a shopping cart that was also maintained through the React Context API. The cart connected to the Shopify API to complete order checkout.
