Jobly

LinkedIn clone


Jobly Code

Mock Job Search

Jobly is a concept app to mimic the structure and capabilities of LinkedIn, including user account creation and login, and company/job search. The website is built with a Express backend and a Postgres database. Frontend for the website built through React with components styled by reactstrap.

Create User Accounts

Users can create accounts to look through companies and jobs to apply for. User signup and login are validated with JSON Schema, while password hashing and authentication are processed with bcrypt. Users can also update their profiles if correct password is entered


Search Companies, Apply to Jobs

Users can search for companies and jobs, as well as list all jobs from a company. Once they find a job they are interested in, users can apply to it with the click of a button.
