Warbler

Twitter clone


Warble Code

Mock Social Network

Warbler is a concept app to mimic the structure and capabilities of Twitter, including user account creation, login, posts, follows, and likes. The website is built with a Flask backend, along with a Postgres database. Data is accessed and managed through SQLAlchemy ORM. Frontend for the website is templated through Jinja.

Create User Accounts

Users can create accounts and profiles to interact with one another. User signup and login forms created and validated with WTForms. Account creation supported with password hashing through bcrypt.


Message Post, Follow Users, Like Posts

Users can post messages on their feed, which can then be viewed by other users. Users can also search for and follow/unfollow other warblers. Followed warblers show up in user's Following list. Users can also like and unlike posts from those followed warblers. Liked posts will show up in user's Like list
